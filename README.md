# 📚 Stack Implementation in Java (Array & LinkedList)

A custom implementation of the **Stack** data structure in Java using both:

- Array-Based Stack
- LinkedList-Based Stack

This project is built using **Object-Oriented Programming (OOP)** principles to demonstrate how stacks work internally without relying on Java's built-in stack implementations.

---

## 💡 What is a Stack?

A **Stack** is a linear data structure that follows the **LIFO (Last In, First Out)** principle:

> The last element added to the stack is the first one to be removed.

---

## ✨ Features

### 📦 Array-Based Stack

- ➕ Push element
- ➖ Pop element
- 👀 Peek top element
- 📊 Display stack elements
- ⚠️ Handle stack overflow and underflow

### 🔗 LinkedList-Based Stack

- ➕ Push element
- ➖ Pop element
- 👀 Peek top element
- 📊 Display stack elements
- ⚡ Dynamic size without fixed capacity
